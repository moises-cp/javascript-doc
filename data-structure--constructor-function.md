# Constructor Function



## Examples

```javascript
function User(firstName, lastName, age, gender) {
	this.firstName = firstName;
    this.lastName = lastName;
    this.age = age;
    this.gender = gender;
}
```



```javascript
const user1 = new User('John', 'Smith', 26, 'male');
```

```javascript
const user2 = new User('Jill', 'Robinson', 32, 'female');
```
